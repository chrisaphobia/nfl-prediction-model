# NFL Game Outcome Prediction Model

A logistic regression model that predicts NFL game winners using historical team performance data, built as a hands-on project to apply machine learning fundamentals.

## Overview

This project pulls real NFL game data (2022–2025 seasons), engineers predictive features — including rolling team performance and season-to-season carryover stats — and trains a classification model to predict whether the home team will win. The goal was to build a complete, working ML pipeline end-to-end while learning core concepts in data preparation, feature engineering, and model evaluation.
